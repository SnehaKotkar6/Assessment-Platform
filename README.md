# Assessment Platform

## Project Description
Assessment Platform is a web-based coding assessment system developed using Java and Spring Boot. It allows users to participate in coding tests, submit solutions, and manage assessments.

## Technologies Used
- Java
- Spring Boot
- Maven
- HTML
- CSS
- JavaScript
- MySQL

## Features
- User Authentication
- Coding Test Management
- Question Management
- Code Submission
- Admin Dashboard
- Result Evaluation

## Project Structure
- src/main/java → Backend source code
- src/main/resources → Configuration and static files
- templates → HTML templates

## How to Run
1. Clone the repository
2. Open project in IntelliJ or Eclipse
3. Configure database in `application.yml`
4. Run the Spring Boot application
